# Rail_Yatri
RAIL YATRI: Online Railway Ticket Booking System
This project is a web application that facilitates the online booking of railway tickets, providing a convenient and efficient way for users to plan their train journeys. It aims to automate the traditional ticket booking process and offer real-time information to users, such as train schedules and seat availability.
Features
User Authentication: Secure user registration and login functionality using login.html and signup.html.
Train Search & Schedule: Users can search for trains between specified source and destination stations and view detailed timetables, including arrival/departure times and distances.
Ticket Booking: A streamlined booking process that allows authenticated users to select trains, check seat availability, and book tickets through search.html and payment.html.
Passenger Management: Functionality to input and manage passenger details (name, age, gender) during the booking process via passengers.html.
Food Ordering: Integration for ordering food during the journey through the food.html interface.
Booking History & Ticket Printing: Users can view their past booking history and print their e-tickets from ticket.html.
Styling & Interactivity: A user-friendly interface is ensured using CSS (styles.css) and interactive elements are managed by JavaScript (script.js).

Usage
New users: Navigate to the Sign Up page (signup.html) to create an account.
Registered users: Use the Login page (login.html) to access their account.
Booking a ticket: After logging in, use the Search feature (search.html) to find trains and proceed to booking.
Managing details: The Profile (profile.html) and Passengers (passengers.html) sections allow for managing personal and travel details.
